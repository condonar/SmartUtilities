# Умные утилиты
Набор консольных утилит на Go, реализующих четыре практических задания: калькулятор, анализ частоты слов, пересечение срезов и журнал посещений врачей.

## Структура проекта
- `Task_1` — консольный калькулятор с поддержкой базовых арифметических операций.
- `Task_2` — поиск K наиболее часто встречающихся слов в строке.
- `Task_3` — поиск пересечения двух неупорядоченных списков целых чисел.
- `Task_4` — интерактивный журнал посещений пациентов (память).

## Задания
1. Консольный калькулятор
   Пошаговый ввод двух чисел (float64) и операции (`+`, `-`, `*`, `/`) с проверкой корректности ввода и форматированным выводом результата (точность деления — 3 знака).
2. Наиболее часто встречающиеся слова
   Чтение строки слов и числа K. Вывод K слов, отсортированных по убыванию частоты, при равенстве частот — лексикографически.
3. Пересечение срезов
   Чтение двух списков целых чисел (int), вывод общих элементов в порядке первого вхождения в первый список. При пустом пересечении — сообщение «Empty intersection», при некорректных данных — «Invalid input».
4. Журнал посещений
   Интерактивный режим с командами:
   - `Save` — сохранение визита (ФИО, специализация врача, дата YYYY-MM-DD).
   - `GetHistory` — история всех посещений пациента.
   - `GetLastVisit` — дата последнего визита к указанному специалисту.
   - Ошибка при отсутствии пациента: `ErrNotFoundVisit: not found visit`.
